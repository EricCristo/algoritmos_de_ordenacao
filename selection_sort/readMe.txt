Que, da mesma forma que criamos “algoritmos mentais” para encontrar o menor valor de uma lista, 
fazemos um processo parecido para ordenar uma lista;

Como podemos abstrair os passos que fazemos “de cabeça” para código, observando a 
sequência dos passos utilizados;

Que, para resolver um problema, muitas vezes utilizamos mais de um algoritmo; 
para resolver isso, criamos a função menorValor() para reaproveitar o código junto com o Selection Sort.
